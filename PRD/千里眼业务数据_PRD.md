# 需求背景
## 需求背景
千里眼业务作为政企客户的重要视频监控业务，涉及大量的终端安装、移机、维护以及故障处理工作。目前，业务数据与财务结算数据之间缺乏直观的关联展示，导致管理人员难以实时掌握各分公司的业务规模、结算费用以及故障报障情况。为了提升业财一体化管理水平，需要建立一套完整的千里眼业务数据管理模块，实现业务量与费用的透明化管理。

## 业务场景
1. **场景一：装移机费用结算**
   管理人员需要按月查看全省及各地市的千里眼装机和移机业务量，并根据预设的单价自动计算应结工料费，支持下钻查看每一笔业务的详细工单信息。
2. **场景二：维护量监控与结算**
   管理人员需要掌握存量千里眼终端的维护规模，按月核算维护费用，确保维护工作的投入与产出清晰可查。
3. **场景三：报障质量分析**
   通过对不同渠道（400热线、10086、APP等）的报障量进行统计，分析故障原因占比，帮助运维团队定位业务薄弱环节，提升服务质量。
4. **场景四：费用标准维护**
   财务或管理人员需要根据业务发展需要，灵活维护不同地市、不同业务类型、不同设备分类的结算单价。

## 业务对象
1. **装移机记录**：包含工单编号、客户信息、设备型号、安装地址、业务类型（装/移）及完成时间。
2. **维护记录**：包含产品实例标识、客户信息、设备序列号、MAC地址及所属地市。
3. **报障工单**：包含报障渠道、故障原因分类、处理环节、处理人及工单历时。
4. **费用单价**：定义特定地市和业务分类下的结算标准。

# 需求概述
本模块旨在提供千里眼业务的业财一体化视图，通过集成装移机、维护、报障等多维度数据，结合灵活的单价配置，实现业务量的自动统计与费用的精准核算。模块包含四个核心子功能：装移机数据管理、维护量数据管理、报障工单数据管理、费用单价管理。

## 功能说明
### 全局布局与导航
**界面元素:**
1. **左侧导航栏**
   - 控件类型: 侧边菜单栏
   - 菜单项: 装移机数据管理、维护量数据管理、报障工单数据管理、费用单价管理。
   - 功能: 支持点击切换子功能视图，支持侧边栏展开/折叠。
2. **顶部页签栏**
   - 控件类型: 页签 (Tabs)
   - 功能: 支持多子功能视图同时打开，支持页签切换、关闭及右键菜单操作（关闭当前、关闭其他、关闭所有）。

### 装移机数据管理
#### 查询筛选区域
**界面元素:**
1. **月份**
   - 控件类型: 月份选择器
   - 默认值: 当前月份 (如 2026-03)
2. **地市**
   - 控件类型: 下拉选择框
   - 选项列表: 全部、内蒙古各盟市
3. **一级分类**
   - 控件类型: 下拉选择框 (只读)
   - 默认值: "千里眼"
4. **二级分类**
   - 控件类型: 下拉选择框
   - 选项列表: 全部、室内无线WIFI、室内外有线
5. **三级分类**
   - 控件类型: 下拉选择框
   - 选项列表: 全部、标准版、专业版、企业版
6. **查询/重置按钮**
   - 功能: 执行筛选或恢复默认条件。

#### 数据列表区域
**表格列定义:**
| 序号 | 列标题 | 数据字段 | 说明 |
|---|---|---|---|
| 1 | 地市 | city | |
| 2 | 业务类型 | businessType | 装机、移机、维护 |
| 3 | 一级分类 | cat1 | |
| 4 | 二级分类 | cat2 | |
| 5 | 三级分类 | cat3 | |
| 6 | 单价 (元) | unitPrice | |
| 7 | 操作 | - | 修改、删除 |

**操作逻辑详细描述:**
- **新增单价**: 点击底部“新增单价”按钮，弹出 **【新增/编辑费用单价弹窗】**。
- **修改**: 点击行内修改图标，弹出 **【新增/编辑费用单价弹窗】** 并回显数据。
  - **【新增/编辑费用单价弹窗】详细描述**:
    - **界面元素**:
      1. **地市** - 必选 (下拉框)
      2. **业务类型** - 必选 (下拉框: 装机、移机、维护)
      3. **一级分类** - 必选 (下拉框: 千里眼)
      4. **二级分类** - 必选 (下拉框)
      5. **三级分类** - 必填 (文本框)
      6. **单价 (元)** - 必填 (数字框)
    - **业务规则**: 同一地市、业务类型、分类组合下仅允许存在一条有效单价记录。
- **删除**: 点击删除图标，弹出二次确认框，确认后物理删除。
- **底部操作按钮详细描述**:
  1. **导入模版**:
     - 控件类型: 按钮 (次要样式)
     - 功能: 点击后下载标准的 Excel 导入模版文件，模版包含地市、业务类型、一级分类、二级分类、三级分类、单价等必填字段说明。
  2. **导入**:
     - 控件类型: 按钮 (次要样式)
     - 功能: 点击弹出文件选择框，支持上传填写好的 Excel 模版文件。系统将校验数据格式及业务规则（如地市是否存在、单价是否为数字），校验通过后批量写入数据库。
  3. **导出**:
     - 控件类型: 按钮 (次要样式)
     - 功能: 点击将当前列表中的所有单价配置数据导出为 CSV/Excel 文件，方便备份或线下核对。
  4. **新增单价**:
     - 控件类型: 按钮 (主要样式)
     - 功能: 点击触发 **【新增/编辑费用单价弹窗】**，用于手动录入单条单价配置信息。

# 非功能性需求
## 性能需求
1. 列表加载时间应小于 2 秒。
2. 下钻详情大数据量加载时应提供加载动画，响应时间不超过 3 秒。
## 安全需求
1. 费用单价管理功能仅限具有“财务管理员”或“系统管理员”权限的用户操作。
2. 导出数据需记录操作日志。
## 可用性需求
1. 界面采用深色科技感风格，关键数值（如费用）使用高亮显示。
2. 弹窗支持自适应屏幕高度，确保在不同分辨率下均可正常操作。
## 兼容性需求
1. 兼容主流浏览器（Chrome, Edge, Firefox）。
