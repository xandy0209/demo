# 需求背景
## 需求背景
随着政企业务的快速发展，终端设备（如千里眼、云视讯、云电脑等）的种类和数量不断增加。为了实现对终端设备全生命周期的精细化管理，需要建立一个统一的“终端库存管理”模块。该模块旨在记录终端设备从入库、出库到拆除的每一个环节，确保库存数据的准确性，提高资产管理效率。

## 业务场景
1. **终端入库录入**：管理人员手动录入或批量导入新到货的终端设备信息，包括SN码、设备类型和厂家。
2. **库存信息查询**：通过SN码、设备类型、厂家、状态或入库时间范围，快速定位特定的终端设备。
3. **信息维护与更新**：对录入有误的终端信息进行修改，或对已报废/错误的记录进行单条或批量删除。
4. **数据导入导出**：支持通过标准模板批量导入数据，并支持将查询结果导出为报表，方便线下核对。

## 业务对象
1. **终端库存记录 (Terminal Inventory Record)**：
   - **SN码**：设备的唯一序列号。
   - **设备类型**：千里眼、云视讯、E企组网、云无线、AC-AP、云电脑等。
   - **厂家**：设备生产厂商。
   - **设备状态**：库存、已出库、已拆除。
   - **时间维度**：入库时间、出库时间、拆除时间。

# 需求概述
“终端库存管理”模块提供终端设备的基础信息维护功能。通过结构化的表格展示和多维度的筛选条件，管理人员可以实时掌握终端资产的分布和状态。模块支持单条录入与批量导入导出，兼顾了操作的灵活性与效率。

## 功能说明

### 查询筛选区域
**界面元素:**
1. **SN码**
   - 控件类型: 文本输入框
   - 占位符: "SN码"
   - 功能: 支持SN码模糊搜索。
2. **设备类型**
   - 控件类型: 下拉选择
   - 选项: 全部、千里眼、云视讯、E企组网、云无线、AC-AP、云电脑。
3. **厂家**
   - 控件类型: 文本输入框
   - 占位符: "厂家"
   - 功能: 支持厂家名称模糊搜索。
4. **状态**
   - 控件类型: 下拉选择
   - 选项: 全部、库存、已出库、已拆除。
5. **时间范围**
   - 控件类型: 日期选择器 (两个)
   - 标签文字: "时间:"
   - 功能: 筛选“入库时间”在指定范围内的记录。
6. **操作按钮**
   - **查询**: 执行筛选操作，并重置页码为1。
   - **置回**: 清空所有筛选条件，恢复默认列表。

### 数据列表区域
**表格列定义:**
| 序号 | 列标题 | 数据字段 | 说明 |
|---|---|---|---|
| 1 | 复选框 | - | 用于勾选行进行批量删除 |
| 2 | SN码 | sn | 终端唯一序列号 |
| 3 | 设备类型 | deviceType | |
| 4 | 厂家 | vendor | |
| 5 | 入库时间 | inboundTime | 格式: YYYY-MM-DD HH:mm:ss |
| 6 | 设备状态 | status | 库存、已出库、已拆除 |
| 7 | 出库时间 | outboundTime | 若未出库显示 "-" |
| 8 | 拆除时间 | removedTime | 若未拆除显示 "-" |
| 9 | 导入人 | importer | |
| 10 | 导入人电话 | importerPhone | |
| 11 | 操作 | - | 包含“修改”和“删除”按钮 |

**操作逻辑详述:**
- **修改**: 点击行末“修改”图标，触发 **【修改终端库存信息弹窗】**。
- **删除**: 点击行末“删除”图标，触发 **【确认删除弹窗】**。

### 底部操作区域
**界面元素:**
1. **导入模版**
   - 控件类型: 按钮
   - 功能: 下载标准的终端信息导入Excel模板。
2. **导入**
   - 控件类型: 按钮
   - 功能: 触发本地文件选择，支持批量上传终端数据。
3. **导出**
   - 控件类型: 按钮
   - 功能: 将当前筛选出的列表数据导出为文件。
4. **添加**
   - 控件类型: 主按钮
   - 功能: 点击触发 **【录入终端库存信息弹窗】**。
5. **批量删除**
   - 控件类型: 危险按钮
   - 状态: 未选中行时禁用。
   - 功能: 点击触发 **【确认删除弹窗】**，显示选中的记录总数。
6. **已选择统计**
   - 功能: 实时显示当前已勾选的记录数量。
7. **分页控件**
   - 功能: 支持切换每页显示条数（默认15条）及翻页。

### 弹窗内联描述

#### 【录入/修改终端库存信息弹窗】
- **弹窗标题**: "录入终端库存信息" (新增模式) / "修改终端库存信息" (编辑模式)
- **界面元素**:
  1. **SN码** - 必填
     - 控件类型: 文本输入框
     - 占位符: "请输入SN码"
  2. **设备类型** - 必填
     - 控件类型: 下拉选择
     - 选项: 千里眼、云视讯、E企组网、云无线、AC-AP、云电脑
  3. **厂家**
     - 控件类型: 文本输入框
     - 占位符: "请输入厂家名称"
- **底部操作**:
  - **取消**: 关闭弹窗。
  - **确定**: 校验必填项。新增模式下生成新记录并置顶；编辑模式下更新对应行数据。

#### 【确认删除弹窗】
- **弹窗标题**: "确认删除"
- **界面元素**:
  - 提示文字: "确定要删除选中的 X 条记录吗？此操作不可恢复。"
- **底部操作**:
  - **取消**: 关闭弹窗。
  - **确定删除**: 从系统中移除对应记录，关闭弹窗并刷新列表。

# 非功能性需求
## 性能需求
- 列表查询及分页响应时间应在 1 秒内。
- 支持万级数据的流畅滚动展示。
## 安全需求
- SN码作为唯一标识，在录入时应进行重复性校验（业务逻辑层）。
- 删除操作必须经过二次确认。
## 可用性需求
- 表格支持横向滚动，确保在低分辨率屏幕下操作列（固定在右侧）始终可见。
- 状态列可根据不同状态显示不同颜色标识（如：库存-绿色，已出库-蓝色）。
