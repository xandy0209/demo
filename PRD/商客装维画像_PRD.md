# 需求背景
## 需求背景
在政企商客业务的运维管理中，装维服务质量（安装与维护）是衡量客户满意度和业务竞争力的关键指标。目前，管理部门对全省装维人员及班组的服务表现缺乏统一、量化的评价工具，导致难以精准定位服务瓶颈或挖掘优秀服务标杆。为了提升装维工作的精细化管理水平，需要构建一套“商客装维画像”系统，通过整合装机量、及时率、满意度、质检通过率等多维度数据，为装维人员提供全方位的数字化评价，支撑管理决策与绩效考核。

## 业务场景
1. **场景一：全省及地市服务质量监控**
   省级管理人员通过概览大屏，实时掌握全省 12 个盟市的装维综合得分分布，识别表现异常的地区。
2. **场景二：装维标杆评选与经验推广**
   通过画像评分排名，筛选出各地区得分最高的 TOP5 装维人员，作为服务标杆进行表彰和经验分享。
3. **场景三：服务短板深度分析**
   管理人员通过趋势分析图表，观察特定地市或个人在及时率、满意度等指标上的月度波动，分析业务下滑原因。
4. **场景四：评价体系动态配置**
   根据业务发展阶段（如阶段性重视质检质量），管理人员动态调整各项指标的计算权重和打分规则，引导装维工作重心。

## 业务对象
1. **装维人员画像**：包含人员基本信息（姓名、班组、归属地市/区县）及核心评价指标（综合得分、装机量、及时率、满意度、质检通过率）。
2. **画像评价指标**：定义画像得分的计算维度，包括指标名称、计算算法、权重比例及具体的打分细则。

# 需求概述
本模块旨在构建商客装维人员的数字化能力模型，通过可视化大屏、明细数据列表和规则配置中心，实现对装维服务质量的全面监控与管理。模块由“商客装维画像概览”、“商客装维画像明细”和“画像配置”三个核心视图组成。

## 功能说明
### 全局布局与导航
**界面元素:**
1. **左侧导航菜单**
   - 控件类型: 侧边导航栏
   - 菜单项: 商客装维画像概览、商客装维画像明细、画像配置。
   - 功能: 点击切换子视图，支持侧边栏收起/展开以优化展示空间。
2. **顶部多页签栏**
   - 控件类型: 页签 (Tabs)
   - 功能: 支持多视图同时打开与快速切换，支持右键菜单（关闭当前、关闭其他、关闭所有）。

### 商客装维画像概览
#### 区域画像得分分布 (核心地图)
**界面元素:**
1. **内蒙古全区地图**
   - 控件类型: ECharts 地图
   - 数据表现: 根据各地市综合得分（80-100分）进行颜色深浅映射。
   - 交互功能: 
     - 悬浮提示：显示地市名称及该地市的平均综合得分。
     - 点击联动：点击地市区域，页面内所有图表及下方的 TOP5 列表同步切换为该地市的数据。
2. **返回全省按钮**
   - 控件类型: 文本按钮
   - 功能: 在选中地市状态下显示，点击后重置地图及所有图表为全省统计视角。

#### 核心指标分析区域 (四组复合图表)
**界面元素:**
1. **装机量 / 装机及时率 / 满意度 / 质检合格率 卡片**
   - 控件类型: 复合图表卡片 (柱状图/折线图)
   - 交互逻辑:
     - **全省视角下**:
       - **地市分布视图 (柱状图)**：点击卡片右上角柱图图标，横轴展示 12 个盟市，纵轴展示对应指标值。
       - **趋势分析视图 (折线图)**：点击卡片右上角折线图标，展示全省最近 6 个月的指标波动趋势。
     - **地市视角下**: 自动切换为该地市最近 6 个月的指标趋势折线图。
2. **装机工单量分布 (环形图)**
   - 控件类型: 环形图 (Donut Chart)
   - 业务分类: 互联网专线、企宽、E企组网、专线卫士、千里眼、云视讯。
   - 功能: 展示各业务类型的工单量占比及中心位置显示工单总数。

#### 商客装维画像 TOP5 列表
**界面元素:**
1. **数据表格**
   - 控件类型: 列表
   - 表格列定义:
| 序号 | 列标题 | 说明 |
|---|---|---|
| 1 | 姓名 | 装维人员姓名 |
| 2 | 班组 | 所属装维班组名称 |
| 3 | 区域 | 归属地市及区县 |
| 4 | 得分 | 综合画像评分 (高亮显示) |
| 5 | 装机量 | 累计完成装机路数 |
| 6 | 及时率 | 装机及时率百分比 |
| 7 | 满意度 | 客户评价满意度百分比 |
| 8 | 质检通过率 | 现场质检合格百分比 |
2. **更多按钮**
   - 功能: 点击跳转至“商客装维画像明细”视图。

### 商客装维画像明细
#### 查询筛选区域
**界面元素:**
1. **地市**
   - 控件类型: 下拉选择框
   - 选项: 全部、呼和浩特市、包头市等 12 个盟市。
2. **区县**
   - 控件类型: 下拉选择框
   - 联动规则: 根据选中的地市动态加载对应的区县列表。
3. **姓名**
   - 控件类型: 文本输入框
   - 占位符: "请输入姓名"
   - 功能: 支持按姓名关键字进行模糊搜索。
4. **查询/重置按钮**
   - 功能: 执行多条件筛选或恢复初始默认状态。

#### 数据列表区域
**表格列定义:**
| 序号 | 列标题 | 数据字段 | 说明 |
|---|---|---|---|
| 1 | 姓名 | name | |
| 2 | 班组 | team | |
| 3 | 地市 | city | |
| 4 | 区县 | district | |
| 5 | 得分 | score | 支持点击表头进行升序/降序排列 |
| 6 | 装机量 | volume | |
| 7 | 及时率 | timeliness | |
| 8 | 满意度 | satisfaction | |

**操作逻辑详细描述:**
- **排序逻辑**: 点击“得分”列标题，支持在“降序”、“升序”、“默认”三种状态间循环切换。
- **分页控件**: 
  - 默认每页显示 20 条。
  - 支持跳转至指定页码。
  - 显示总记录数。
- **导出功能**: 点击底部“导出”按钮，将当前筛选条件下的所有数据导出为 CSV 格式文件（包含 UTF-8 BOM 以防中文乱码）。

### 画像配置
#### 配置状态与全局操作
**界面元素:**
1. **配置状态监控**
   - 控件类型: 状态标签
   - 逻辑: 
     - **已应用生效**：当前列表中的权重总和等于 100% 且已执行应用。
     - **待应用（已修改）**：用户对指标进行了增删改操作，但尚未执行“配置应用”。
2. **配置应用按钮**
   - 功能: 触发权重校验。
   - 业务规则: 遍历所有指标权重，若总和不等于 100%，弹出错误提示并禁止应用；若等于 100%，则更新系统画像计算逻辑并提示成功。
3. **新增指标按钮**
   - 功能: 触发 **【新增画像指标弹窗】**。

#### 指标配置列表
**表格列定义:**
| 序号 | 列标题 | 数据字段 | 说明 |
|---|---|---|---|
| 1 | 指标名称 | name | 如：装机及时率 |
| 2 | 计算算法 | algorithm | 描述具体的数学计算公式 |
| 3 | 权重 (%) | weight | 该指标在总分中所占的百分比 |
| 4 | 打分规则 | scoringRule | 详细的阶梯打分或扣分逻辑描述 |
| 5 | 操作 | - | 包含“编辑”和“删除”功能 |

**操作逻辑详细描述:**
- **编辑操作**: 点击行内编辑图标，触发 **【编辑画像指标弹窗】**。
  - **【新增/编辑画像指标弹窗】详细描述**:
    - **界面元素**:
      1. **指标名称** (必填): 文本框。
      2. **计算算法** (必填): 文本框。
      3. **权重 (%)** (必填): 数字输入框，范围 0-100。
      4. **打分规则** (必填): 多行文本输入框。
    - **底部操作**: 点击“保存配置”更新列表数据，状态切换为“待应用”。
- **删除操作**: 点击行内删除图标，触发 **【删除确认弹窗】**。
  - **【删除确认弹窗】详细描述**: 提示“确定要删除该指标配置吗？此操作不可撤销”，确认后移除记录，状态切换为“待应用”。

# 非功能性需求
## 性能需求
1. 概览页面所有 ECharts 图表应支持响应式缩放，在窗口调整大小时无错位。
2. 列表分页切换响应时间应在 500ms 以内。
## 安全需求
1. 画像配置功能必须具备严格的权限控制，仅限管理员角色操作。
## 可用性需求
1. 采用深色科技感 UI 风格，关键指标（如得分）需使用霓虹蓝高亮显示。
2. 所有数值类数据（如装机量）应支持千分位展示。
## 兼容性需求
1. 兼容 Chrome 80+、Edge 等主流现代浏览器。
