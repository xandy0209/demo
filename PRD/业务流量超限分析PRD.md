# 需求背景

## 需求背景
（基于系统的当前实现描述）
当前系统需要对业务流量的使用情况进行监控与分析，以便运维人员识别零流量业务及高带宽占用超限业务，提升网络资源的利用效率与异常处置速度。

## 业务场景
- **场景一：流量异常监控**：运维人员通过该模块监控全省及各市的零流量业务占比和高带宽占用超限业务占比。
- **场景二：明细下钻分析**：在发现统计数据异常时，通过点击对应指标进入详情视图，查看具体的客户、终端及流量物理指标明细。

## 业务对象
1. **统计记录对象**：包含时间、地市、业务总数、零流量业务数/占比、高带宽占用超限业务数/占比。
2. **详情明细对象**：包含采集时间、客户信息（名称/编号）、区域信息（省份/地市/安装地址）、业务账号、终端信息（类型/MAC/串号）、流量指标（带宽、上下行流量、利用率、上线时间）。

# 需求概述
“业务流量超限分析”模块是一个集成统计展示与明细下钻功能的管理工具，通过柱形图（侧边栏逻辑）与数据列表结合，实现对业务饱和度的量化管理。

### 业务流量统计分析页面
#### 查询筛选区域
**界面元素:**
1. **统计粒度**
   - 控件类型: 下拉选择框
   - 选项列表: 天、月
   - 默认值: 天
2. **统计周期**
   - 控件类型: 单一日选择器 / 单一月选择器
   - 标签文字: "统计周期:"
   - 默认值: **昨日** (当前日期 - 1天)
   - 交互规则: 当统计粒度为“天”时，提供单日选择控件 (`type="date"`)；当统计粒度为“月”时，提供单月选择控件 (`type="month"`)。为保证数据准确性，本页面采用单点时间分析而非范围选择。切换粒度时将自动清空当前时间。
3. **地市**
   - 控件类型: 下拉选择框 (Select)
   - 选项列表: **全部**、**全区**、呼和浩特、包头、鄂尔多斯等内蒙各地市。
   - 默认值: **全部**
   - 逻辑规则:
     - 选择“全部”时，展示列表包含省级的“全区”汇总记录及其下属所有地市记录；“全区”记录将置顶显示。
     - 选择“全区”时，列表仅展示省级汇总记录。
     - 选择具体地市时，列表仅展示对应地市记录。
4. **操作按钮**
   - **查询**: 执行数据筛选并重置页码。
   - **重置**: 清空筛选条件，并将粒度恢复为“按天”。

#### 数据列表区域
**表格列定义:**
| 序号 | 列标题 | 数据字段 | 说明 |
|---|---|---|---|
| 1 | 时间 | time | 统计日期/月份 |
| 2 | 地市 | city | 所属地市 |
| 3 | 业务总数 | totalBusiness | - |
| 4 | 零流量业务数 | zeroTrafficCount | 点击蓝色的数字触发【详情弹窗】 |
| 5 | 零流量业务占比 (%) | zeroTrafficRatio | - |
| 6 | 高带宽占用超限业务数 | highBandwidthCount | 点击橙色的数字触发【详情弹窗】 |
| 7 | 高带宽占用超限业务占比 (%) | highBandwidthRatio | - |

**操作逻辑详细描述:**
- **数据加载**: 进入页面时展示 Loading 动画，提示“正在采集分析数据...”。
- **详情下钻**: 
  - 点击“零流量业务数”列中的数值，弹出 **【零流量业务详情弹窗】**。
  - 点击“高带宽占用超限业务数”列中的数值，弹出 **【高带宽占用超限业务详情弹窗】**。

| 3 | 客户编号 | customerCode |
| 4 | 省份 | province |
| 5 | 地市 | city |
| 6 | 宽带账号 | broadbandAccount |
| 7 | 终端类型 | terminalType |
| 8 | 终端MAC | terminalMac |
| 9 | 终端串号 | terminalSn |
| 10 | 订购带宽 | orderedBandwidth |
| 11 | 上行流量 | upstreamTraffic |
| 12 | 下行流量 | downstreamTraffic |
| 13 | 上行速率 | upstreamRate |
| 14 | 下行速率 | downstreamRate |

- **底部操作**: 
  - **导出**: 导出当前详情列表数据。

#### 底部操作与分页
**界面元素:**
1. **导出**
   - 控件类型: 底部按钮
   - 功能: 导出主表统计数据。
2. **统计统计信息**
   - 显示当前条件下的统计记录总数。
3. **分页控件**
   - 默认每页显示: 15条
   - 功能: 支持切换页码和调整每页条数。

# 非功能性需求
## 异常处理
- **无数据状态**: 当查询结果为空时，列表居中显示“暂无统计数据”。
- **加载状态**: 详情数据加载过程中呈现 Loading 动画，提示“正在加载详情数据...”。
